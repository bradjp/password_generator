Create a password generator.

You should provide a length, and have a password returned. #

The password can consist of lower and upper case letters, numbers, and the symbols '%' and '$'. #

If no length is provided, return a password of 8 characters. #

If a length below 6 is specified, return 'Insecure password length.' #

If a length above 150 is specified, return 'Too many characters.' #

Extras:

Allow a password to receive only unique characters.

Allow a password to exclude special characters.
